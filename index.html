Featured Projects

Bookstore

2025 Python Django SQLite HTML CSS Bootstrap JavaScript


Bookstore is a full-stack web application that functions as an online bookstore, where users can browse books by category, search using multiple parameters, and manage purchases through a dynamic shopping cart. The platform includes registration, login, pagination, customer comments with CRUD functionality, a contact form, and an embedded Google Map showing the store’s location. The project brings together core e-commerce features into one cohesive platform.

TheatreCMS3

2025 C# .NET MVC EF SQLServer HTML CSS JavaScript Bootstrap


TheatreCMS3 is a full-stack content management system built for a theater website, featuring tools to manage rentals, events, performances, and user interactions. I contributed primarily to the Blog section, developing and styling full-stack CRUD functionality and improving user experience with Bootstrap and JavaScript. I also implemented asynchronous blog deletion and access controls to ensure only authorized users could manage content.

Evently

2025 Python Django SQLite HTML CSS Bootstrap JavaScript JQuery


Evently is a full-stack web application for discovering and managing events of interest. It integrates the Ticketmaster API for event search, supports saving favorites, and lets users add and manage their own events with full CRUD functionality. Additional features include Google Maps integration, a blog section powered by web scraping, and direct ticket purchase links.
